  这是一款比Chrome略小，一款高速安全上网浏览器，其核心与界面由贾科大的EasyWebkit与HoloDesignUI项目组提供，
软件致力于打造一个无后门，无推广，无插件，无广告的公益清爽浏览器。
  由于本版本受到网友青睐，且为了名副其实，所以开放源代码。
  欢迎广大易友，相关专业学者进行学习、指正与二次开发。

 软件主要特色有：
  1.防有声广告，彻底屏蔽洗脑声音功能，避免精神污染或被迷惑 
  2.防下载中毒，彻底屏蔽下载功能 
  3.防国外内核后门，彻底弃用主流Chrome内核，采用自研WKE 
  4.内置FlashPlayer17与HTML5引擎 
  5.防朋友圈恶意脚本链接造成卡死，彻底屏蔽窗口劫持弹窗脚本 
  6.防各类脚本广告，更新至V8版本引擎，彻底屏蔽广告常用脚本功能 
  7.独立内核，独立渲染，与系统隔离，从根本上消除安全隐患
  8.支持WAP页面访问，支持Smart在线QQ，默认UA为通用

 软件更新历程：
  1.0.0.0 基于IE内核的单页浏览器，系统默认UI
  2.0.0.0 基于IE内核的双页浏览器，自行设计UI
  3.0.0.0 基于IE内核打造的多页浏览器，功能丰富，自绘UI
  4.0.0.0 基于WKE自研项目内核，重新设计界面为HoloLight风格，回归专业单页面

  注：目前最新版本已更新到6.0版本，使用类MaterialDesign并搭载最新内核
